# Charts in Excel

## Overview
Charts visualize data for insights and presentations. This guide covers creating and customizing bar, line, and pie charts.

## Key Concepts
- **Chart Types**:
  - **Bar**: Compare categories (e.g., sales by region).
  - **Line**: Show trends (e.g., revenue over time).
  - **Pie**: Display proportions (e.g., market share).
- **Inserting Charts**: Select data, Insert > Chart Type.
- **Customization**:
  - Add titles (Chart Title).
  - Adjust labels (Data Labels).
  - Change colors (Chart Styles).

## Step-by-Step Example
**Scenario**: Create a bar chart for sales by region.
1. **Prepare Data**:
   - Columns: Region (A2:A5), Sales (B2:B5).
2. **Insert Chart**:
   - Select A1:B5.
   - Insert > Bar Chart > Clustered Bar.
3. **Customize**:
   - Add title: “Sales by Region”.
   - Enable Data Labels (Chart Elements > Data Labels).
   - Change bar color (Format > Shape Fill).

## Practical Scenario
